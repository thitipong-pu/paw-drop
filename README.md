# Paw Drop 🐾

A modern web application for managing your pet's drops, appointments, and daily activities — built with the latest web technologies.

## Tech Stack

- **[React 19](https://react.dev/)** — UI Framework
- **[Vite 6](https://vitejs.dev/)** — Build Tool
- **[TypeScript](https://www.typescriptlang.org/)** — Type Safety
- **[Tailwind CSS v4](https://tailwindcss.com/)** — Utility-first CSS
- **[HeroUI React v3](https://heroui.com/)** — Component Library
- **[Framer Motion](https://www.framer.com/motion/)** — Animations

## Getting Started

### Prerequisites
- Node.js 20+
- npm 10+

### Installation

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
# Build for production
npm run build

# Preview the production build
npm run preview
```

## Design System

Paw Drop uses **HeroUI v3** with **Tailwind CSS v4** for a sleek, dark-mode-first interface. Key design principles:

- 🌑 **Dark mode first** — Built on `zinc-950` backgrounds
- 🎨 **Gradient accents** — Pink-to-violet primary palette
- ✨ **Micro-animations** — Smooth transitions via Framer Motion
- 💎 **Premium feel** — Glassmorphism and layered shadows
