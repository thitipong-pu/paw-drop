## 📍 Section 3: Modal รายละเอียดสถานที่

เมื่อกดที่ Place Card จะเปิด Bottom Sheet Modal แสดง:

```
┌──────────────────────────────┐
│ ─── (handle bar)             │
│ [emoji] ชื่อสถานที่          │
│         [badge ประเภท]       │
├──────────────────────────────┤
│ 📍 ที่อยู่เต็ม               │
│ [ปุ่ม 📋 คัดลอกที่อยู่]      │  ← copy to clipboard
│ [ปุ่ม 🗺️ เปิดใน Google Maps] │
├──────────────────────────────┤
│ 📞 โทรศัพท์  |  ⏰ เวลาทำการ │
├──────────────────────────────┤
│ 🧡 สิ่งที่ต้องการ            │
│ [chip] [chip] [chip]...      │
└──────────────────────────────┘
```

### ฟีเจอร์ปุ่มคัดลอก
- กดปุ่ม → `navigator.clipboard.writeText(address)`
- ปุ่มเปลี่ยนเป็น ✅ "คัดลอกแล้ว!" สีเขียว 2 วินาที
- Fallback: ใช้ `execCommand('copy')` สำหรับเบราว์เซอร์เก่า

## 📍 Section 4: สินค้าที่ต้องการบริจาค

### Category Tabs
- **ทั้งหมด** — แสดงทุกรายการ
- **🐕 สุนัข** — กรองเฉพาะสุนัข
- **🐈 แมว** — กรองเฉพาะแมว
- **🐾 อื่นๆ** — กระต่าย นก เต่า
- **🔴 ด่วน!** — สินค้าที่ต้องการเร่งด่วน

### Product Card
```
┌─────────────┐
│  🐕 emoji  │  ← icon สินค้า (พื้น gradient ส้มอ่อน)
├─────────────┤
│ 🔴 ต้องการด่วน  (ถ้ามี)
│ ชื่อสินค้อ  │
│ แบรนด์      │
│ [🛍️ Shopee] │  ← ปุ่มสีแดง Shopee
└─────────────┘
```

### การเปิด Shopee
- กดที่ Product Card หรือปุ่ม Shopee → `window.open(shopeeSearchUrl, '_blank')`
- URL format: `https://shopee.co.th/search?keyword={ชื่อสินค้า}`

## 🎯 UX Principles ที่ใช้

1. **Mobile-first** — ออกแบบสำหรับมือถือก่อน ใช้งานง่ายด้วยนิ้วเดียว
2. **Progressive disclosure** — แสดงข้อมูลสำคัญก่อน รายละเอียดเพิ่มเติมใน modal
3. **One-tap action** — ทุก action สำคัญทำได้ในคลิกเดียว
4. **Emotional design** — สีอบอุ่น emoji สัตว์น่ารัก กระตุ้นความรู้สึกอยากช่วย
5. **Urgency indicators** — badge "ต้องการด่วน" กระตุ้นการตัดสินใจ
